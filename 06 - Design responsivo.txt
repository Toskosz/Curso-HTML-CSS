* Agora precisamos deixar nosso site responsivo, isto é, que ele funcione tanto no navegador quanto no celular e o primeiro passo deve ser adicionar a seguinte tag no <head>:
<meta name="viewport" content="width=device-width">

* Então no css:
@media screen and (max-width: 480px) {
	.caixa, .principal, .conteudo-beneficios, .mapa-conteudo, .video {
		width: auto;
	}
	h1 {
		text-align: center;
	}
	nav {
		position: static;
	}
	.lista-beneficios, .imagem-beneficios {
		width: 100%;
	}
}

* Não ficou muito claro o @media, aparentemente não mudou nada e o resultado final não foi o mesmo do vídeo, depois pesquisar mais afundo sobre responsividade 

* Qual a principal tática para não perder muito tempo criando layouts responsivos?
	Usar medidas proporcionais para tudo e Criar um layout fluido que se adapte sempre
